JS — sync aria-expanded (optional but recommended)

const det = document.getElementById('winDetails'); det.addEventListener('toggle', () => { det.querySelector('summary').setAttribute('aria-expanded', det.open); });

Why this beats a custom

  • Native semantics — screen readers announce "disclosure widget, collapsed/expanded" automatically
  • Zero JS required for basic open/close — works without JS, enhanced with aria-sync
  • Keyboard native — Enter/Space toggles, Tab moves in/out correctly
  • Print/Reader mode safe — content stays accessible when styles stripped
  • Find-in-page works — hidden content still searchable when closed (browser-dependent but usually yes)
